Question 1 Problem: How does one shift ALL elements in an array 1 index to the left, leaving the last element null? 
For example: The array
A
B
C
D
E
F

becomes
B
C
D
E
F
null


Create an IPO+S Chart to analyse this problem.
Input:
Input array

Processing:
Initialize working array to length of input array

Iterate through the working array starting at second element until end of array and for each index, set the working array at this index-1 to the value of the input array at the current index

Then set last element in working array to NULL

Output:
A newly shifted array

Storage:
A second working array


Write pseudocode for a function named shiftArrayLeft() that will solve this problem.
```
FUNCTION shiftArrayLeft(arr)
    let workingArr = new Array(arr.length);
    FOR i from 1 to arr.length-1
        SET workingArr[i-1] = arr[i]
    NEXT i
    SET workingArr[arr.length-1] = NULL

    RETURN workingArr
END FUNCTION
```
